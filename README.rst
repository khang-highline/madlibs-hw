Building on Window
==================

Dependency
----------

* make (must be added to path)
* Cygwin or MSYS (must be added to path)
* Visual Studio 2022

Compile
-------

Open the project in Developer Command Prompt for VS 2022.

Then, run the command:

.. code-block::

	make
